Rock Paper Scissors (JavaScript)

A simple Rock Paper Scissors game built with JavaScript as part of my learning process.

🚀 Features
- User input using prompt
- Random computer choice
- Game logic for determining winner
- Score tracking over 5 rounds
- Input validation (only accepts rock, paper, or scissors)

🧠 What I Learned
- How to break problems into pseudocode
- Converting pseudocode into working JavaScript
- Writing functions and organizing code
- Using loops to control game flow
- Debugging common JavaScript errors
- Handling user input safely

🕹️ How to Play
- Run the script in a browser (DevTools console)
    Enter:
    "rock"
    "paper"
    or "scissors"
- Play 5 rounds against the computer
- Final score will be displayed in the console

🛠️ Technologies Used
- JavaScript

📂 How to Run
- Open browser (Chrome recommended)
- Open DevTools (F12)
- Paste the code into the console
- Run playGame()
